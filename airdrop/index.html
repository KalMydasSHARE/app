Comment fonctionne la distribution ?

  • 1.Les bénéficiaires sont enregistrés par l'équipe KalMydas
  • 2.Vos KAL sont automatiquement mis en jeu dans le contrat de récompenses — ils génèrent des rewards dès le premier jour
  • 3.Le se fait par tranches : 25% immédiat, puis 25% tous les 30 jours (100% après 90 jours)
  • 4.Quand vous réclamez, vous recevez le principal débloqué + les rewards en une seule transaction
  • 5.Les KAL réclamés arrivent dans votre portefeuille — vous pouvez les garder, les revendre, ou les remettre en mise en jeu
